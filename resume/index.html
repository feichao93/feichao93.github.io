temme

从 HTML 中提取 JSON 数据的开源工具。在 CSS 选择器的基础上使用 PEG.js 定义了额外的语法来抓取 HTML 中所需要的字段，支持多字段选取、列表选取、格式处理等功能。 该工具包含了完善的文档、测试、实例，设置了持续集成并发布在 NPM 上。 同时 VSCode 编辑器中也有该工具的相应插件，提供了语法高亮和执行选择器的功能。 项目地址： https://github.com/shinima/temme

little-saga

根据 redux-saga 源码修改而来的简化版本。little-saga 核心部分并不绑定 redux，而是实现了独立的 effects 处理机制与 fork model。 little-saga 具有良好的拓展性，可以在运行时更新 effect 的解释器，为新的 effect 类型注册对应的处理函数。 little-saga 核心部分只包括少数 effect 类型（promise / fork / join / cancel 等）， 剩下的 effect 类型（all / race / take / put 等）则使用拓展的方式进行实现，并跑通了 redux-saga 中相应的测试。 项目地址： https://github.com/shinima/editor
在知乎上我也写了一篇非常详细的文章介绍 little-saga 的原理。 https://zhuanlan.zhihu.com/p/37356948

简易 SVG 编辑器 (2018.01 - 2018.04)

该编辑器主要功能是绘制多边形等形状，以及将语义信息附加到绘制的形状上。 编辑器提供了丰富的快捷键和自动对齐功能，可以完成较为复杂的图形绘制。 编辑器采用了响应式编程框架 Cycle.js，对键盘、鼠标、用户行为等进行了合理抽象，用简洁直观的代码清晰地描述出了应用的数据流。 该编辑器是 TRIPS 系统 的室内空间建模工具；而 TRIPS 论文已经被 VLDB 2018 接受。 项目地址： https://github.com/shinima/editor
